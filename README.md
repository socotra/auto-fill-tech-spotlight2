# Auto-Fill Demo Configurations

## Configs

* `auto-fill-basic-plugin`: basic plugin-based implementation
* `auto-fill-policy-copy`: plugin-based implementation of a policy rewrite/copy technique 

## Postman

These routines are provided to exercise the functionality in the configs. It also includes routines to set up an external service if you wish, by first (1) setting up the external service integration, then (optionally) (2) specifying the call sequence by using the auto-fill integration endpoint.
